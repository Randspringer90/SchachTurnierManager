Companion-Modus

Mit dem Turnier-PC verbinden

Diese App zeigt Turniere, Paarungen und Tabellen von einem laufenden SchachTurnierManager auf deinem PC. Der PC muss im selben WLAN erreichbar sein.

Die Adresse steht im PC-Fenster beim Start (z. B. „http://192.168.…:5088"). Keine feste Adresse fest eingebaut – du gibst sie selbst ein.

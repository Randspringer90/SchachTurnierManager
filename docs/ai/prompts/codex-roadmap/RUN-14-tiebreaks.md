# RUN-14 – Tie-Breaks / Wertungen FIDE-nah härten

Vorab `PROMPT_BASE.md` lesen und befolgen.

## Ziel
Kampflose Partien, Byes und ungespielte Runden in allen Wertungen FIDE-nah behandeln.

## Aufgaben
- Offenen PLANS-Punkt „Buchholz-Feinheiten, kampflose Partien, Cut-Wertungen sauber
  spezifizieren" abschließen: zuerst Spezifikation als Doku (FIDE-Regelstand angeben),
  dann Implementierung.
- Virtual-Opponent-Modell (FIDE-Regel für ungespielte Partien in Buchholz & Co.)
  fertig verdrahten und testen.
- Buchholz, Buchholz Cut-1/Cut-2, Median, Sonneborn-Berger, Direktvergleich, Siege,
  Performance: je Wertung Testfälle mit Bye/kampflos/Rückzug.
- Wertungskette im UI verständlich konfigurierbar machen (verständliche Labels,
  Tooltips/Erklärungen, sinnvolle Presets).
- Golden Tests für Beispielturniere mit bekannten Endständen ergänzen.

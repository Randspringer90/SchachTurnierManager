# Entscheidungsregister (Decisions)

> Kurze ADR-artige Eintraege. Dateiname: `NNNN-titel.md`. Pflichtmetadaten: `source`, `date`,
> `trust`, `review`. Beispielkopf:

```
source: intern (PR #NN) | date: 2026-07-12 | trust: T2 | review: owner-approved
```

Nur gepruefte Entscheidungen; keine PII/Secrets; untrusted Inhalte werden nicht als Regel abgelegt.
